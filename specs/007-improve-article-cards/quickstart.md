# Quickstart: Improve Timeline Article Cards

## Prereqs
- Node.js 20+
- Dependencies installed (`npm install`)

## Run
```bash
npm run dev
```

## Verify
1. Log in and open the timeline.
2. Confirm each card shows title link, metadata line, excerpt, and optional right-side thumbnail.
3. Click the title link; it should open in a new tab without expanding the card.
4. Click the card body to expand; it should mark the item read while staying visible.
5. In expanded view, confirm title + metadata at top, full-width thumbnail (if any), and full text or excerpt fallback.
